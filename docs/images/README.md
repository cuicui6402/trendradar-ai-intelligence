## 截图目录

该目录用于存放 TrendRadar 项目运行截图。

需要补充的截图：

- `home.png` — 首页与分析设置：展示关键词输入框、项目说明卡片和侧边栏
- `result.png` — 趋势评分与排行榜：展示 KPI 指标卡、GitHub/Hacker News 表格和趋势评分柱状图
- `report.png` — AI 趋势报告预览：展示生成的 9 章节 Markdown 报告
- `history.png` — 历史记录回看：展示点击历史记录后从数据库恢复的分析结果

截图建议：使用浏览器全屏宽度，Streamlit 页面 `layout=wide` 模式，关键词示例推荐 "AI Agent"。
